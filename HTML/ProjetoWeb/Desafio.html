O que são Tags

(Voltar)

Cabeça do HTML, aqui daremos o título da página usando a tag (< title>)

Corpo do HTML, aqui será exibido ao usuário oque foi desenvolvido na página Meu Primeiro HTML

Tag (< i >) Deixa o texto em itálico (Texto em itálico)

Tag (< strong >) Deixa o texto em negrito (Texto em Negrito)

Tag (< strong > < i >) Deixa o texto negrito itálico (Texto em Negrito Itálico)

Tag (< strong > < u >) Deixa o texto negrito sublinhado (Texto Negrito sublinhado)

Tag (< mark >) Marca o texto (Texto Marcado)

Tag (< input >) Cria um campo de imputação de dados (text=texto) (number=numérico) (color=paleta de cores)

Aqui foi utilizado o input com o atributo (type)para definir o tipo de dado que será inserido no campo, Onde solicitamos o NOME com o atributo (name), Utilizado o atributo (placeholder) para criar uma mensagem orientativa ao usuário, Trazendo o seguinte resultado.

Tag (< id >) identificador do elemento (id="titulo" Meu primeiro html)

Tag (< style >) utilizada no CSS para formatação, Aqui atribuímos o azul ao texto onde passamos o atributo (style), com o parâmetro (color: blue) Texto formatado em azul

tag (< class >) utilizada no CSS para formatação em classe com esse atribuído, conseguimos definir um tipo de elemento no qual será aplicada a todos os scripts que possuírem esse nome na classe, atributo (class="titulo da classe") Atributo class

tag (< img >) utilizada para inserir imagem, atributo (width) utilizado para definir a largura da imagem, seguido de (src) no qual inserimos o link ou diretório de onde esta a imagem

Tipos de Listas

(Voltar)

Listas Ordenadas e Não Ordenadas

Listas Ordenadas

Criando uma lista ordenada (Lista em ordem numérica)

Tag (< ol >) que e definida pela tag (< li >).

Exemplo da estrutura de lista

< ol > < li>Item < / li > < / ol >

Resultado
  1. Item 1
  2. Item 2
  3. Item 3

    4. Criando lista dentro de outra lista. Aqui vemos a possibilidade de criar listas dentro de listas utilizando a tag (< ol >) dentro da tag ( < li >)

    Exemplo da estrutura de lista

    < ol> < li>Item < ol > < li>Item principal.subitem < / li > < /ol > < / li > < / ol >

  4. Item 4 com subitens
    1. Item 4.1
    2. Item 4.2
    3. Item 4.3

    5. Listas Não Ordenadas

    Criando uma lista não ordenada (Lista com marcadores)

    Exemplo da estrutura de lista

    < ul> < li>Item < / li> < / ul>

    Resultando
    • Item 1
    • Item 2
    • Item 3
