O que são Tags

(Voltar)

Cabeça do html, aqui daremos o titulo da pagina usando a tag (< title >)

Corpo do html, aqui será exibido ao usuário oque foi desenvolvido na pagina Meu primeiro html

Tag (< i >) Deixa o texto em italico (Texto em Italico)

Tag (< strong >) Deixa o texto em negrito (Texto em Negrito)

Tag (< strong > < i >) Deixa o texto negrito italico (Texto em Negrito Italico)

Tag (< strong > < u >) Deixa o texto negrito sublinhado (Texto Negrito sublinhado)

Tag (< mark >) Marca o texto (Texto Marcado)

Tag (< input >) Cria um campo de imputação de dados (text=texo) (number=numerico) (color=paleta de cores)

Aqui foi utilizado o input com o atributo (type)para definir o tipo de dado que será inserido no campo, Onde solicitamos o NOME com o atributo (name), Utilizado o atributo (placeholder) para criar uma mensagem orientativa ao usuário, Trazendo o seguinte resultado.

Tag (< id >) identificador do elemento (id="titulo" Meu primeiro html)

Tag (< style >) utilizada no css para formatação, Aqui atribuimos a cor azul ao texto onde passamos o atributo (style), com o parametro (color: blue) Texto formatada em azul

tag (< class >) utilizada no css para formatação em classe com esse atribudo, conseguimos definir um tipo de elemento no qual será aplicada a todos os scripts que possuirem esse nome na classe, atributo (class="titulo da classe") Atributo class

tag (< img >) utilizada para inserir imagem, atributo (width) utilizado para definir a largura da imagem, seguido de (src) no qual inserimos o link ou diretorio de onde esta a imagem

Tipos de Listas

(Voltar)

Listas Ordenadas e Não Ordenadas

Listas Ordenadas

Criando uma lista ordenada (Lista em ordem numerica)

Tag (< ol >) que e definida pela tag (< li >).

Exemplo da estrutura de lista

< ol > < li >Item < / li > < / ol >

Resultando
  1. Item 1
  2. Item 2
  3. Item 3

    4. Criando lista dentro de outra lista. Aqui vemos a possibilidade de criar listas dentro de listas utilizando a tag (< ol >) dentro da tag (< li >)

    Exemplo da estrutura de lista

    < ol > < li >Item < ol > < li >Item principal.subitem < / li > < /ol > < / li> < / ol >

  4. Item 4 com subitens
    1. Item 4.1
    2. Item 4.2
    3. Item 4.3

    5. Listas Não Ordenadas

    Criando uma lista não ordenada (Lista com marcadores)

    Exemplo da estrutura de lista

    < ul > < li >Item < / li > < / ul >

    Resultando
    • Item 1
    • Item 2
    • Item 3
